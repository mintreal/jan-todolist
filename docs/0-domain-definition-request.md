너는 최고의 비즈니스 도메인 전문가야. "Jan TodoList"라는 이름의 사용자 인증 기반 TODO LIST 애플리케이션을 하려고 해. 다음 내용을 바탕으로 적절한 서브에이전트를 활용해 도메인 정의서 초안을 작성해줘.

# 문제 정의 :

- 사용자마다 할일 목록을 관리할수 있는 앱이 있으면 좋겠다.
- 할일을 캘린더 형태로도 볼수 있는 탭이 있으면 좋겠다
- 할일이 만료기간이 표현되어 기한내 처리할 수 있도록 강조되어있음 좋겠다

# 앱의 주요 기능 :

- 회원가입 및 회원의 로그인/로그아웃 기능
- 무료 가입 기능
- 회원의 할일이 리스트업되고, 수정 및 삭제, 추가 기능
- 기한이 임박한 할일에 대해 상단에 노출되고, 지정된 기한이 임박하면 알림의 기능

# 필수 사항 :

- 도메인 정의서에는 핵심적인 내용만 포함하도록 함
- 작성된 도메인 정의서를 docs/ 디렉토리에 1-domain-definition.md 파일로 저장함
